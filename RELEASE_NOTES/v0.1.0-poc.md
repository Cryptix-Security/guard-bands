# v0.1.0-poc

Initial proof-of-concept release for Guard Bands.

This release packages the core FastAPI POC, HMAC-based Guard Band wrapping and verification, SSO-aware audit metadata, pytest coverage for core security scenarios, and CI/release automation.

Notable hardening in this release:

- Canonical context serialization for signing and verification.
- Nonce values included in authenticated MAC input.
- Fail-closed chat behavior when guard-banded content is not verified before final response.
- Expanded replay-protection and tool-call enforcement examples.

This is a research POC and is not production-ready without deployment-specific key management, replay ledgers, authorization, and operational hardening.
