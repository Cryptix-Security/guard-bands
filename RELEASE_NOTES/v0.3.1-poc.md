# v0.3.1-poc

Dependency and release hygiene update for Guard Bands.

This patch release publishes the maintenance changes that landed after `v0.3.0-poc`.

Notable changes:

- Bumped pinned runtime dependencies:
  - `anthropic` from `0.111.0` to `0.112.0`
  - `fastapi` from `0.138.0` to `0.138.1`
- Updated stale test and benchmark marker literals to match the `v0.3.0` Guard Band wire format.
- Added a manual `workflow_dispatch` trigger to GitHub Actions CI.

Security status at release:

- GitHub Actions CI passes on Python 3.11 and 3.12.
- CodeQL passes.
- Open CodeQL alerts: none.
- Open Dependabot alerts: none.

This remains a proof-of-concept release. Production use still requires environment-specific key management, authorization design, shared replay storage for multi-replica deployments, TLS, audit retention, monitoring, and operational review.
